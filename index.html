Car Price Analysis and Predictive Modeling

In this project I created a presentation and a predictive model to understand which features affect car prices, and to properly identify the price range of a potential customer's dream car. My goal involved the use of several techonologies in order to create a powerful presentation and obtain an accurate Machine Learning model. In the end I found some crucial characteristics when it comes to car price, such as Year manufactured, Mileage, and Engine Size. These characteristics provided the basis for an accurate regression model. I was then able to successfully identify the price range of a dream car for a potential customer.


Key technologies and methodologies used in this project include:
    • Python
    • Machine Learning - Scikit-Learn
    • Data Exploration & Data Transformation - Pandas, Numpy, Normalization
    • Data Visualization - Matplotlib, Seaborn
    • Feature Engineering - Scikit-Learn
    • Ensemble Methods
    • Predictive Modeling

Building A Deep Learning Library From Scratch

The creation of 'Micrograd' has provided me with a deep understanding of the inner workings of libraries used for deep learning and neural network development, such as PyTorch. Micrograd can perform complex calculations to obtain neural network outputs and utilizes backpropagation to update gradients of weights and biases. Then, by calculating the loss using Sum Square Error (SSE) we can reduce the loss and improve the model accuracy.
Key features and methodologies include:

    • Deep Learning Libraries (PyTorch)
    • Neural Network Construction
    • Backpropagation for Gradient Descent
    • Loss Function Optimization (Sum of Squared Errors)
    • Predictive Accuracy Enhancement
